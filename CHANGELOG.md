# Changelog

## 0.1.0

- Initial native Windows migration.
- Direct2D GPU rendering and Windows Imaging Component decoding.
- Fast wheel navigation and pointer-anchored `Control` + wheel zoom.
- Stable top-left window anchoring across image aspect-ratio changes.
- Direction-aware preview prefetching and bounded memory cache.
- Borderless normal viewing and immersive fullscreen background fill.
- Per-user installer scripts and Windows Default apps registration.
- Portable x64 ZIP, checksum, CI, and release automation.
